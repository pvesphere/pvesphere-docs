# 存储管理

存储管理功能允许您监控和管理 Proxmox VE 集群中的存储资源。

## 存储列表

存储列表显示所有集群中的存储信息：

- **存储名称**: 存储的标识符
- **类型**: 存储类型（local/nfs/ceph 等）
- **集群/节点**: 存储所属的集群和节点
- **容量**: 总容量和可用容量
- **使用率**: 存储使用百分比

## 存储操作

### 查看存储详情

点击存储名称可以查看详细信息：

- 存储配置
- 容量统计
- 使用情况
- 性能指标

### 浏览存储内容

可以浏览存储中的内容：

- **ISO 镜像**: 安装镜像文件
- **备份文件**: 虚拟机备份
- **模板**: VM 模板文件
- **容器模板**: LXC 容器模板

### 存储操作

- **上传文件**: 上传 ISO 镜像或其他文件
- **删除文件**: 删除不需要的文件
- **下载文件**: 下载存储中的文件

## 存储类型

### Local Storage

本地存储，直接连接到节点：

- 性能高
- 容量受节点限制
- 不支持跨节点共享

### NFS Storage

网络文件系统存储：

- 支持跨节点共享
- 适合共享模板和备份
- 需要网络连接

### Ceph Storage

分布式存储系统：

- 高可用性
- 自动数据冗余
- 适合大规模部署

## 监控和告警

### 容量监控

实时监控存储容量：

- 总容量
- 已使用容量
- 可用容量
- 使用趋势

### 告警通知

当存储容量不足时，系统会发送告警：

- 容量阈值告警
- 存储故障告警
- 性能问题告警

## 最佳实践

1. **容量规划**: 定期检查存储容量，提前规划扩容
2. **备份策略**: 将重要数据备份到共享存储
3. **性能优化**: 根据使用场景选择合适的存储类型
4. **监控告警**: 设置合理的容量告警阈值
