# 虚拟机管理

虚拟机管理是 PveSphere 的核心功能之一，提供了完整的虚拟机生命周期管理。

## 虚拟机列表

虚拟机列表显示所有集群中的虚拟机：

- **名称**: 虚拟机的名称和 ID
- **集群/节点**: 虚拟机所在的集群和节点
- **状态**: 运行状态（Running/Stopped）
- **资源**: CPU、内存使用情况
- **IP 地址**: 虚拟机的 IP 地址

## 虚拟机操作

### 创建虚拟机

1. 点击 "+Create VM" 按钮
2. 选择目标集群和节点
3. 配置虚拟机参数：
   - 名称和描述
   - 操作系统类型
   - CPU 和内存
   - 磁盘大小
   - 网络配置
4. 选择安装方式（ISO/模板）
5. 配置 Cloud-Init（可选）
6. 点击 "Create" 创建虚拟机

### 启动/停止虚拟机

- **启动**: 启动已停止的虚拟机
- **停止**: 优雅地停止运行中的虚拟机
- **强制停止**: 强制关闭虚拟机
- **重启**: 重启虚拟机

### 虚拟机控制台

通过 VNC/NoVNC 访问虚拟机控制台：

1. 点击虚拟机操作菜单中的 "Console"
2. 在浏览器中打开 VNC 控制台
3. 可以像本地桌面一样操作虚拟机

### 虚拟机迁移

支持在线迁移虚拟机到其他节点：

1. 选择目标节点
2. 选择迁移模式（在线/离线）
3. 确认迁移参数
4. 执行迁移

### 备份和恢复

- **创建备份**: 备份虚拟机的完整状态
- **恢复备份**: 从备份恢复虚拟机
- **备份管理**: 查看和管理所有备份

## 配置管理

### 硬件配置

可以修改虚拟机的硬件配置：

- CPU 核心数和类型
- 内存大小
- 磁盘大小和类型
- 网络适配器
- 其他硬件设备

### Cloud-Init 配置

支持 Cloud-Init 配置：

- 用户账户
- SSH 密钥
- 网络配置
- 自定义脚本

## 监控和日志

### 资源监控

实时监控虚拟机资源使用：

- CPU 使用率
- 内存使用率
- 磁盘 I/O
- 网络流量

### 任务日志

查看虚拟机的操作历史：

- 创建记录
- 配置变更
- 迁移历史
- 备份记录
