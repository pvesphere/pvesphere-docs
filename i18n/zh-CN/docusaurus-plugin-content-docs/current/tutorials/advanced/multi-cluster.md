# 多集群管理

PveSphere 支持管理多个 Proxmox VE 集群，实现统一的资源管理。

## 集群组织

### 环境分类

建议按环境对集群进行分类：

- **test**: 测试环境集群
- **dev**: 开发环境集群
- **prod**: 生产环境集群

### 区域划分

可以按地理位置划分集群：

- **SH**: 上海区域
- **BJ**: 北京区域
- **GZ**: 广州区域

## 统一管理

### 全局视图

在仪表板中可以查看所有集群的全局状态：

- 资源汇总
- 健康状态
- 告警信息

### 跨集群操作

支持跨集群的操作：

- 虚拟机迁移
- 模板同步
- 备份管理

## 最佳实践

1. **命名规范**: 使用统一的命名规范
2. **权限控制**: 为不同环境设置不同的访问权限
3. **监控告警**: 为所有集群配置监控和告警
4. **备份策略**: 制定统一的备份策略
