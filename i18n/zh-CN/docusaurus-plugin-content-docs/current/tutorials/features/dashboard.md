# 仪表板

仪表板提供了 PveSphere 系统的全局概览，帮助您快速了解所有集群、节点、虚拟机和存储的状态。

## 功能概览

### 资源统计

仪表板顶部显示关键资源统计：

- **集群总数**: 已添加的 PVE 集群数量
- **节点总数**: 所有集群中的物理节点数量
- **虚拟机总数**: 所有集群中的虚拟机数量
- **存储总量**: 所有存储的总容量

### 资源利用率

实时显示资源利用率指标：

- **CPU 使用率**: 所有节点的 CPU 使用情况
- **内存使用率**: 所有节点的内存使用情况
- **存储使用率**: 所有存储的使用情况

### 热点和风险

#### Top Resource Consumers

显示资源消耗最高的实体：

- **VM**: 按 CPU 或内存使用率排序的虚拟机
- **Node**: 按资源使用率排序的节点
- **Storage**: 按使用量排序的存储

#### Recent Risks (24h)

显示最近 24 小时内的风险告警：

- 资源告警
- 健康检查失败
- 性能问题

### 多集群切换

可以通过环境筛选器切换不同环境的集群视图：

- **All**: 显示所有集群
- **test**: 仅显示测试环境集群
- **dev**: 仅显示开发环境集群
- **prod**: 仅显示生产环境集群

## 使用技巧

1. **实时刷新**: 点击刷新按钮更新最新数据
2. **筛选查看**: 使用环境筛选器查看特定环境的资源
3. **快速导航**: 点击资源卡片快速跳转到对应的管理页面
