# 节点管理

节点管理功能允许您监控和管理 Proxmox VE 集群中的物理节点。

## 节点列表

节点列表显示所有集群中的节点信息：

- **节点名称**: 节点的标识符
- **集群**: 节点所属的集群
- **状态**: 节点的运行状态（Online/Offline）
- **CPU**: CPU 使用率和核心数
- **内存**: 内存使用情况
- **存储**: 存储使用情况

## 节点操作

### 查看节点详情

点击节点名称可以查看详细信息：

- 硬件信息
- 资源使用情况
- 网络配置
- 服务状态

### 节点控制台

通过终端代理访问节点控制台：

1. 点击节点操作菜单中的 "Console"
2. 在终端窗口中执行命令
3. 支持完整的终端功能

### 服务管理

可以管理节点上的服务：

- **启动服务**: 启动指定的服务
- **停止服务**: 停止指定的服务
- **重启服务**: 重启指定的服务

### 网络配置

查看和配置节点网络：

- 网络接口信息
- IP 地址配置
- 路由表
- 防火墙规则

## 监控功能

### 资源监控

实时监控节点资源：

- CPU 使用率
- 内存使用率
- 磁盘 I/O
- 网络流量

### 告警通知

当节点出现问题时，系统会发送告警：

- 资源使用过高
- 服务异常
- 连接失败
