# 监控与告警

PveSphere 提供了全面的监控和告警功能。

## 监控指标

### 资源监控

- **CPU**: CPU 使用率和负载
- **内存**: 内存使用情况
- **存储**: 存储容量和 I/O
- **网络**: 网络流量和带宽

### 性能监控

- **响应时间**: API 响应时间
- **任务执行**: 任务执行状态
- **错误率**: 错误发生频率

## 告警配置

### 告警规则

可以配置告警规则：

- **阈值告警**: 资源使用超过阈值
- **状态告警**: 服务状态异常
- **性能告警**: 性能指标异常

### 告警通知

支持多种通知方式：

- 邮件通知
- 短信通知
- Webhook 通知

## 监控面板

### 仪表板

在仪表板中可以查看：

- 实时监控数据
- 历史趋势图表
- 告警信息汇总

### 自定义监控

可以创建自定义监控面板：

- 选择监控指标
- 配置图表类型
- 设置刷新频率
