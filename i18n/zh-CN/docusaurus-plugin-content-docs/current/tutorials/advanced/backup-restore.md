# 备份与恢复

PveSphere 提供了完整的备份和恢复功能。

## 备份策略

### 备份类型

- **完整备份**: 备份虚拟机的完整状态
- **增量备份**: 只备份变更的部分
- **差异备份**: 备份与上次完整备份的差异

### 备份计划

可以创建自动备份计划：

- 备份频率（每日/每周）
- 备份时间
- 保留策略
- 目标存储

## 恢复操作

### 完整恢复

从备份完整恢复虚拟机：

1. 选择备份文件
2. 选择目标集群和节点
3. 配置恢复参数
4. 执行恢复

### 部分恢复

可以只恢复部分数据：

- 磁盘恢复
- 配置文件恢复
- 数据文件恢复

## 最佳实践

1. **定期备份**: 制定定期备份计划
2. **测试恢复**: 定期测试恢复流程
3. **异地备份**: 将备份存储到异地
4. **版本管理**: 维护备份版本历史
